# ATDF102-grupo-8
Repositorio del grupo 8 para el ramo Introducción a la Programación (ATDF102). Aquí se registran los avances, tareas, ejercicios y proyectos desarrollados en equipo durante el semestre.

## to run

1. Clone the repository:
   ```bash
   git clone https://github.com/belcaik/ATDF102-grupo-8.git
   ```
2. Navigate to the project directory:
   ```bash
   cd ATDF102-grupo-8
    ```

3. run main.py from root directory:
   ```bash
   python src/main.py
   ```
